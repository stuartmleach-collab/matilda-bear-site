# Matilda Bear — website (master copy)

This folder is the live Matilda Bear website: six pages, one shared stylesheet
(`styles.css`), one small script (`app.js`), two printable PDFs, and an `assets`
folder of images and fonts. Everything the site needs is here.

- **Live at:** https://matildabear.co.uk
- **Hosted on:** Netlify
- **Contact form:** handled by Netlify Forms (messages appear in Netlify's Forms tab
  and are emailed to the site owner)

To make a change: edit the files here, then publish to Netlify. For a full plain-English
explanation of the whole setup, see `handbook.md` in the folder above this one.
